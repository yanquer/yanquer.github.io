创建项目

修改于: 2023-02-20

命令行:

django-admin startproject $项目名

应用创建

命令行:

python manage.py startapp $应用名

然后在 settings.py 中的 INSTALLED_APPS 中添加应用名.

比如创建一个名为 polls 的应用:

python manage.py startapp polls

然后在 settings.py 中的 INSTALLED_APPS 中添加 polls.

INSTALLED_APPS = [
    'django.contrib.admin',
    'django.contrib.auth',
    'django.contrib.contenttypes',
    'django.contrib.sessions',
    'django.contrib.messages',
    'django.contrib.staticfiles',

    # 新增的应用
    'polls.app.PollsConfig',
]

接着创建数据表

 # 检测对模型文件的修改（在这种情况下，你已经取得了新的），并且把修改的部分储存为一次 迁移。
 python manage.py makemigrations polls
 # 第一次执行会将建表语句储存在 polls/migrations/0001_initial.py
 # 可选, 查看对应的sql语句
 # python manage.py sqlmigrate polls 0001

 python manage.py migrate

# 重置数据库
 # python manage.py flush

注解

一个项目可以包含多个应用, 每个应用可以独立运行.

如果需要新增模型:

python manage.py startapp 模型名称
