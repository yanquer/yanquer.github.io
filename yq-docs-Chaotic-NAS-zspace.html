虚拟机smb挂载

  • Windows: Windows电脑通过SMB挂载极空间的方法
  • Linux: linux如何通过smb挂载极空间

主要说下Linux的

  1. 安装smbclient
  2. 准备账密

示例的IP地址：192.168.1.227

示例用户名：18600000000

  1. 在linux客户端上执行命令，输入密码，可以查询可挂载的目录：
smbclient -L 192.168.1.227 -U 18600000000
#Enter WORKGROUP\18600000000's password:
#    Sharename       Type      Comment
#    ---------       ----      -------
#    共享文件    Disk
#    186XXXX0000     Disk
#    Time Machine    Disk
  1. 在步骤3中选择需要挂载的路径名称，拼接下面的命令。如有其他高级应用需求，请自行搜索参数，拼接命令。
# 创建本地路径
mkdir 本地路径
# 执行挂载命令
mount.cifs -o username=18600000000,password="密码" //192.168.1.227/186XXXX0000 本地路径
  1. 验证挂载是否成功
mount | grep cifs
