electron-log

修改于: 2024-07-21
相关资源
  • npm地址: https://www.npmjs.com/package/electron-log
  • github地址: https://github.com/megahertz/electron-log
  • 主页: https://github.com/megahertz/electron-log#readme

日志使用, 支持 node, 主进程, 渲染进程

安装

npm install electron-log --save

支持的调用级别:

error, warn, info, verbose, debug, silly

主进程使用

import log from 'electron-log/main';

// Optional, initialize the logger for any renderer process
log.initialize();

log.info('Log from the main process');

渲染进程使用

import log from 'electron-log/renderer';
log.info('Log from the renderer process');

纯node使用

import log from 'electron-log/node';
log.info('Log from the nw.js or node.js');

设置格式与获取位置

// 默认
//  '%c{h}:{i}:{s}.{ms}%c › {text}' (main)
//  '{h}:{i}:{s}.{ms} › {text}'
log.transports.console.format = '{h}:{i}:{s} {text}';

log.transports.file.getFile();

设置输出通道

不输出到文件/控制台

log.transports.file.level = false;
log.transports.console.level = false;

输出文字颜色

log.info('%cRed text. %cGreen text', 'color: red', 'color: green')

支持的颜色

  • unset (reset to default color)
  • black
  • red
  • green
  • yellow
  • blue
  • magenta
  • cyan
  • white

捕获错误(Catch errors)

支持直接设置自动捕获未处理的错误

log.errorHandler.startCatching(options?);

多个日志实例

您可以创建具有不同设置的多个记录器实例：

import log from 'electron-log/main';

const anotherLogger = log.create({ logId: 'anotherInstance' });

注解

需要单独配置每个实例（例如日志文件路径）。

日志范围(Logging scopes)

import log from 'electron-log/main';
const userLog = log.scope('user');

userLog.info('message with user scope');
// Prints 12:12:21.962 (user) › message with user scope
