react-select

修改于: 2024-06-13

好一点的 Select 下拉框实现

为什么有这个? 因为原生的 select 不支持设置样式...

相关资源
  • https://www.npmjs.com/package/react-select
  • https://react-select.com/home
  • https://github.com/JedWatson/react-select

安装

yarn add react-select

说明

  • 除了同步的 Select, 还有异步的 import AsyncSelect from 'react-select/async';

    异步组件可配请求前的临时数据 defaultoptions , 参考: https://react-select.com/async#defaultoptions

  • value 优先级高于 defaultValue, 即当同时设置时, 页面显示选中的是 value 的值

使用

import * as React from 'react';
import Select from 'react-select'

interface opOne {
    label: string,
    value: string,
}
const options: opOne[] = [
    { value: 'chocolate', label: 'Chocolate' },
    { value: 'strawberry', label: 'Strawberry' },
    { value: 'vanilla', label: 'Vanilla' }
]

const MySelectComponent = () => (
    // false 表示不允许多选
    <Select<opOne, false>
        options={options}
        isMulti={false}
        isDisabled={false}
        onChange={(val, actionMeta) => console.log(val)}
        placeholder={"请输入..."}
        autoFocus={false}
        isSearchable={false}
        menuPortalTarget={document.body}
        theme={theme => ({
            ...theme,
            borderRadius: 0,
        })}
    />
)
