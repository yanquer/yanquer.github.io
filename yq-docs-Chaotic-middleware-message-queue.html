消息队列

修改于: 2024-03-09

rabbitmq地址: AMQP 0-9-1 Protocol

此文主要介绍 AMQP (Advanced Message Queuing Protocol) 协议: 高级消息队列协议

设计目标: 让服务端可通过协议编程

组成:

  • 网络协议
  • 服务端服务

包含:

  • 高级消息队列协议模型(AMQ Model)
  • 网络层协议AMQP, 让客户端程序与实现了AMQ Model的服务端进行通信

AMQP协议是一个二进制协议，具有一些现代特性:

  • 多通道（multi-channel）
  • 可协商（negotiated）
  • 异步
  • 安全
  • 便携
  • 语言中立
  • 高效

基本的AMQP模型:

              |   Exchange          Queue     |
发布者 ->      |    交换机    ->    Msg Queue   |  -> 消费者
Publisher     |                               |     Consumer
              |         AMQP Model            |

消息由交换机路由给消息队列
