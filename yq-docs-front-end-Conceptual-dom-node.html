Dom节点

修改于: 2024-07-18

HTMLElement

类似于元素基类

常用属性/方法

innerHTML
获取或设置元素的 HTML 内容
outerHTML
获取或设置元素的 HTML 内容, 包括元素本身
innerText
获取或设置元素的文本内容
textContent
获取或设置元素的文本内容, 包括所有子节点的文本内容
querySelector()
返回匹配指定选择器的第一个元素
querySelectorAll()
返回匹配指定选择器的所有元素
getAttribute()
获取元素的指定属性值
setAttribute()
设置元素的指定属性值
removeAttribute()
移除元素的指定属性
classList

返回元素的类名, 作为 DOMTokenList 对象

可以使用 add()、remove() 和 toggle() 方法添加、删除和切换类名

也可以使用 contains() 方法检查类名是否存在 适用于需要操作元素类名的场景

appendChild()
在元素的子节点列表末尾添加新的子节点
removeChild()
删除元素的子节点
insertBefore()
在指定子节点前插入新的子节点
replaceChild()
替换元素的子节点
cloneNode()

复制元素

可以传入一个布尔值参数, 表示是否复制元素的所有子节点

注解

像 insertBefore 这样的操作, 如果插入的节点已经存在于 DOM 中, 那么会先从原来的位置删除, 然后再插入到新的位置, 即移动元素的操作

HTMLDivElement

支持属性

clientWidth/clientHeight
  • 包括元素的内容区域和左右内边距的宽度，不包括边框和滚动条。 (元素内容区域宽度 + 左右内边距宽度)
offsetWidth/offsetHeight
  • 包括元素的宽度、左右边框和左右滚动条（如果存在）的宽度。 (元素宽度 + 左右边框宽度 + 左右滚动条宽度)
offsetTop/offsetLeft
  • 返回元素相对于其父元素的偏移量
scrollWidth/scrollHeight
  • 包含元素内容的完整宽度/高度,包括因内容超出而隐藏的部分。
  • 即使元素有滚动条,也会返回完整的内容尺寸。
  • 适用于需要获取元素完整内容尺寸的场景。
scrollLeft/scrollTop
  • 获取或设置元素的水平/垂直滚动条的位置。
  • 返回值是元素滚动条距离左上角的距离。
  • 适用于需要获取或设置元素滚动位置的场景。
getBoundingClientRect()
  • 返回一个对象,包含元素的尺寸和相对于视口的位置。
  • 返回的对象包含 left、top、right、bottom、width、height 等属性。
  • 适用于需要获取元素相对于视口的位置信息的场景。
getComputedStyle()
  • 返回一个对象,包含元素最终的样式值,包括从样式表、内联样式或 CSS 属性计算得到的值
  • 适用于需要获取元素最终计算后的样式属性值的场景。

HTMLCanvasElement

HTMLCanvasElement.captureStream()

返回的 CanvasCaptureMediaStream 是一个实时视频捕获的画布。 注意需要有画布操作, 否则blob大小为0

参考: https://developer.mozilla.org/zh-CN/docs/Web/API/HTMLCanvasElement/captureStream
