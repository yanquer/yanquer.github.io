div

fieldset

一般作为组织表单元素集合使用. legend 内容可以当做标题使用.

<fieldset>
  <legend>表单标题</legend>
  <label>
    <input type="checkbox" name="checkbox" value="1">
    <span>复选框</span>
  </label>
  <label>
    <input type="radio" name="radio" value="1">
    <span>单选框</span>
  </label>
</fieldset>

fieldset坑-会被table子元素的宽度撑大

参考: <pre>会撑开父辈<fieldset>的宽度，怎么能让pre的宽度随filedset的宽度

这是因为 fieldset 在没有内部元素时，是以 block 方式解析的，即 外部100%宽度自适应 ；

原因

min-width 属性影响， 在CSS的属性的权重中， min-width > width

在chrome里面:

fieldset { min-width: -webkit-min-content; }
解决方案
手动设置 min-width
fieldset {
  min-width: 100%;
}

或者

fieldset {
  min-width: inherit;
}

pre

常用与包裹代码段

pre被子元素撑大

参考: <pre>会撑开父辈<fieldset>的宽度，怎么能让pre的宽度随filedset的宽度.

pre元素为何会自动撑宽。

因为其 white-space 属性:

pre { white-space: pre; }

pre 的语义既是预保留格式，这意味着（外部标签无法决定 pre 标签的宽度， pre 标签自动包裹内部的文本节点，不忽略空白//有误，见下）。

pre 标签的宽度是由如下机制进行的

  • 在内部内容不宽过父容器时，其宽度会自适应于父容器；
  • 在内部内容比父容器宽的时候，其宽度为包裹内部内容。

这个机制类似于 min-width ，但比 min-width 还多了对于换行/空白符的处理，下面是 w3school 对于 white-space 属性值的解释：

  • normal 默认。空白会被浏览器忽略。
  • pre 空白会被浏览器保留。其行为方式类似 HTML 中的 <pre> 标签。
  • nowrap 文本不会换行，文本会在在同一行上继续，直到遇到 <br> 标签为止。
  • pre-wrap 保留空白符序列，但是正常地进行换行。
  • pre-line 合并空白符序列，但是保留换行符。
  • inherit 规定应该从父元素继承 white-space 属性的值。

把 white-space 改为除了 nowrap 和 pre 以外的值即可符合你的要求。不过我想问，你真的需要修改 pre 标签的默认行为吗？改变 pre 标签的默认行为，即意味着你放弃了 pre 标签的原本语义。

详细说明

svg是 SVG 图形的根容器

<svg width="200" height="100" viewBox="0 0 200 100">
  <!-- SVG内容 -->
</svg>

其中, width/height 可以由外部指定, viewBox 则是 SVG 的坐标系大小, 实际会按照宽高比进行缩放

linearGradient 线性渐变

<defs>
  <linearGradient id="grad1" x1="0%" y1="0%" x2="100%" y2="0%">
    <stop offset="0%" style="stop-color:rgb(255,0,0);stop-opacity:1" />
    <stop offset="100%" style="stop-color:rgb(255,255,0);stop-opacity:1" />
  </linearGradient>
</defs>
<rect x="10" y="10" width="180" height="80" fill="url(#grad1)"/>

radialGradient 径向渐变

<defs>
  <radialGradient id="grad2" cx="50%" cy="50%" r="50%" fx="50%" fy="50%">
    <stop offset="0%" style="stop-color:rgb(255,255,255);stop-opacity:0" />
    <stop offset="100%" style="stop-color:rgb(0,0,255);stop-opacity:1" />
  </radialGradient>
</defs>
<ellipse cx="100" cy="50" rx="85" ry="30" fill="url(#grad2)"/>

pattern 图案填充

<defs>
  <pattern id="pattern1" patternUnits="userSpaceOnUse" width="20" height="20">
    <rect width="10" height="10" fill="red"/>
    <rect x="10" y="10" width="10" height="10" fill="red"/>
  </pattern>
</defs>
<rect width="200" height="100" fill="url(#pattern1)"/>

filter 滤镜效果

<defs>
  <filter id="blur">
    <feGaussianBlur in="SourceGraphic" stdDeviation="5"/>
  </filter>
</defs>
<rect x="20" y="20" width="160" height="60" fill="blue" filter="url(#blur)"/>

image 绘制图像

<image x="20" y="20" width="160" height="120" xlink:href="image.jpg"/>

a 超链接

<a xlink:href="https://example.com">
  <rect x="50" y="20" width="100" height="50" fill="orange"/>
</a>

animate 基础动画

<rect x="10" y="10" width="50" height="50" fill="red">
  <animate attributeName="x" from="10" to="150" dur="3s" repeatCount="indefinite"/>
</rect>

animateTransform 变换动画

<rect x="20" y="20" width="50" height="50" fill="blue">
  <animateTransform attributeName="transform" type="rotate" from="0 45 45" to="360 45 45" dur="5s" repeatCount="indefinite"/>
</rect>

path 路径绘制

<path d="M 10 80 Q 95 10 180 80 T 350 80" stroke="black" fill="transparent"/>
路径命令详解

路径命令由字母（表示命令类型）和坐标参数组成，命令字母大小写敏感：

  • 大写字母表示绝对坐标
  • 小写字母表示相对坐标

移动命令 (M/m)

<path d="M x,y 后续命令..."/>
  • M (Move to): 将画笔移动到指定点，不画线
  • 示例: M 10,20 移动到绝对坐标 (10,20)
  • 示例: m 10,20 相对于当前位置移动 (10,20)

直线命令 (L/l, H/h, V/v)

<path d="M x,y L x,y" stroke="black"/>
  • L (Line to): 从当前点到指定点画直线
  • H (Horizontal): 水平线，只需x坐标
  • V (Vertical): 垂直线，只需y坐标
  • 示例: L 50,50 画到绝对坐标 (50,50)
  • 示例: l 30,0 向右画30px水平线

闭合命令 (Z/z)

<path d="M10,10 L50,10 L30,50 Z"/>
  • Z: 闭合路径，从当前点画直线回起点
  • 不区分大小写，不需要参数

三次贝塞尔曲线 (C/c, S/s)

<path d="M10,80 C40,10 65,10 95,80" stroke="black" fill="none"/>
C (Cubic Bézier):
  • 语法: C x1,y1 x2,y2 x,y
  • (x1,y1): 起点控制点
  • (x2,y2): 终点控制点
  • (x,y): 终点
S (Smooth cubic Bézier): 简化的连续曲线
  • 语法: S x2,y2 x,y
  • 自动对称前一个控制点

二次贝塞尔曲线 (Q/q, T/t)

<path d="M10,80 Q52.5,10 95,80" stroke="black" fill="none"/>
Q (Quadratic Bézier):
  • 语法: Q x1,y1 x,y
  • (x1,y1): 控制点
  • (x,y): 终点
T (Smooth quadratic Bézier): 简化的连续曲线
  • 语法: T x,y
  • 自动对称前一个控制点

椭圆弧曲线 (A/a)

<path d="M10,50 A30,30 0 0 1 50,50" stroke="black" fill="none"/>
A (Arc):
  • 语法: A rx,ry x-axis-rotation large-arc-flag sweep-flag x,y
  • rx, ry: x轴和y轴半径
  • x-axis-rotation: 椭圆旋转角度(度)
  • large-arc-flag: 0表示小弧，1表示大弧
  • sweep-flag: 0表示逆时针，1表示顺时针
  • x,y: 终点坐标
