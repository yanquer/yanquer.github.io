代理模型

修改于: 2025-11-09

代理模型是Django模型的一种特殊类型, 它允许你创建一个新的模型类, 该类继承自现有的模型类, 但不会创建新的数据库表. 代理模型主要用于修改现有模型的行为, 例如添加自定义方法,更改默认排序等, 而不需要更改数据库结构.

例子

# models.py
from django.db import models

class MyModel(models.Model):
    name = models.CharField(max_length=100)
    type = models.CharField(max_length=20, choices=[('type1', '类型1'), ('type2', '类型2')])
    status = models.CharField(max_length=20, choices=[('active', '激活'), ('inactive', '未激活')])
    created_at = models.DateTimeField(auto_now_add=True)

    class Meta:
        ...

# 第一个代理模型 - 显示类型1的数据
class Type1Model(MyModel):
    class Meta:
        proxy = True
        verbose_name = '类型1数据'
        verbose_name_plural = '类型1数据'

# 第二个代理模型 - 显示类型2的数据
class Type2Model(MyModel):
    class Meta:
        proxy = True
        verbose_name = '类型2数据'
        verbose_name_plural = '类型2数据'
