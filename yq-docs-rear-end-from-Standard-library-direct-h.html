mkdir

参考: 创建目录_mkdir()函数

创建文件夹

原型

int mkdir(const char *path,mode_t mode);
  • 函数名: mkdir
  • 功 能: 建立一个目录
  • 用 法: int mkdir( const char *dirname );
  • 头文件库：direct.h
  • 返回值：创建一个目录，若成功则返回0，否则返回-1

如 mkdir("test", 0777);

opendir

参考: 打开、读取以及关闭目录[ opendir()、 readdir()和 closedir() ]

打开一个目录，并返回指向该目录的句柄，供后续操作使用

函数原型

DIR *opendir(const char *name);

readdir

读取目录，获取目录下所有文件的名称以及对应 inode 号

函数原型

struct dirent *readdir(DIR *dirp);

rewinddir

将目录流重置为目录起点，以便对 readdir() 的下一次调用将从目录列表中的第一个文件开始

函数原型

void rewinddir(DIR *dirp);

closedir

关闭处于打开状态的目录，同时释放它所使用的资源

函数原型

int closedir(DIR *dirp);
