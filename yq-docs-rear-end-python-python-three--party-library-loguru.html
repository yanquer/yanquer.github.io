loguru库是一个用于日志记录的库, 提供了简单易用的API和丰富的功能.

相对于 logging 库, loguru提供了更多的功能和易用性。

安装:

pip install loguru

基本使用

from loguru import logger

logger.debug('this is a debug message')
logger.info('this is an info message')

添加输出渠道, 比如文件/标准输出

from loguru import logger

# 每1MB分割一个日志文件
file_log = logger.add('runtime.log', rotation='1 MB')
