list 求交集并集差集

不建议的:

# 假设有两个集合 a,b
# 交集
[val for val in a if val in b]
# 并集
list(set(a+b))
# 差集
[val for val in b if val not in a]    # b中有而a中没有的

建议的高效的:

# 假设有两个集合 a,b
# 交集
list(set(a).intersection(set(b)))
# 并集
list(set(a).union(set(b)))
# 差集
list(set(b).difference(set(a)))       # b中有而a中没有的
