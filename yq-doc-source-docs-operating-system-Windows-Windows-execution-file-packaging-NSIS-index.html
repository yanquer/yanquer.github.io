nsis打包

参考文档::NSIS 打包脚本基础

NSIS（Nullsoft Scriptable Install System）是一个开源的 Windows 系统下安装程序制作程序

大体流程是:

  • 编写 NSIS 脚本(后缀为 nsi 的文件)
  • 编译打包. 默认会打包 NSIS 脚本所在目录的所有文件/目录. 可使用GUI界面或者: index
