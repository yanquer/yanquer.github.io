line-height

修改于: 2024-06-21

注解

举个栗子, 当你发现 input 内文字的位置偏上/下, 多半是行高不对.

还可能影响两个控件的对齐行为, 比如在行高一致的时候, 两个组件无其他样式时候能正常对齐, 不一致的时候就无法对齐...

参考
源于AI

line-height 和 height 是 CSS 中两个不同的属性，它们用于控制元素的尺寸和文本的布局，但作用不同。

height
  • 定义：height 属性用于设置元素的高度。
  • 效果：它直接影响元素的块级模型，设定元素的可视高度。

用法:

div {
    height: 100px; /* 设置高度为100像素 */
}
line-height
  • 定义：line-height 属性用于设置文本行之间的垂直间距。
  • 效果：它影响文本的垂直对齐和行间距，但不直接改变元素的高度（除非没有其他高度设置）。

用法:

p {
    line-height: 1.5; /* 行高为1.5倍字体大小 */
}
主要区别
作用范围：
  • height 针对整个元素的高度。
  • line-height 针对文本的行距，影响每行文本的高度。
使用场景：
  • height 通常用于控制盒子模型的高度。
  • line-height 用于改善文本的可读性和美观性。

示例

div {
    height: 100px; /* 设定整体高度 */
    line-height: 50px; /* 文本行间距 */
}

在这个例子中，div 的高度是 100 像素，而文本的行高是 50 像素。这意味着如果 div 中有多行文本，它们之间的间距会是 50 像素，但整体 div 的高度仍然是 100 像素。
