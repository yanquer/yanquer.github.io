插入视频

正常的markdown插入视频有以下方式

方式1:

<video width="100%" height="100%" controls autoplay muted>
<source src="xxx/xxx.mp4?raw=true" type="video/mp4">

方式2:

<video src="xxx/xxx.mp4?raw=true">

方式3:

<video src="xxx/xxx.mp4?raw=true">

方式4:

![观看这个视频](xxx/xxx.mp4?raw=true)

方式5:

<!--嵌套中放入一张类似视频播放界面的图片诱导用户点击跳转链接从而播放视频，当然这不是一个正确的方式
[![Watch the video](xxx.png)](xxx/xxx.mp4?raw=true)
-->

github插入视频

参考
  • https://blog.csdn.net/weixin_45940369/article/details/130551018?spm=1001.2101.3001.6650.3&utm_medium=distribute.pc_relevant.none-task-blog-2%7Edefault%7EBlogCommendFromBaidu%7ECtr-3-130551018-blog-136309360.235%5Ev43%5Epc_blog_bottom_relevance_base6&depth_1-utm_source=distribute.pc_relevant.none-task-blog-2%7Edefault%7EBlogCommendFromBaidu%7ECtr-3-130551018-blog-136309360.235%5Ev43%5Epc_blog_bottom_relevance_base6
  • https://github.com/heartyang520

github不支持外部视频链接或者存储库视频直接调用, 可以用issue上传的视频文件链接, 多个视频展示留空一行, 注意且mp4格式最大10M

此方案注意要在github上的视频是可预览的:

<details>
    <summary>
        <h3>Open Up</h3>
    </summary>

xxx/xxx.mp4?raw=true
</details>

注解

自己用的时候发现直接放在resource里面的mp4没被自动识别为视频, 然后这个方案就不行.
