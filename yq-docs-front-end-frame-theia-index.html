Theia

修改于: 2024-03-09

一个IDE框架, 前后端分离的架构

以插件的形式编写代码, 支持依赖注入

后端可选浏览器/Electron

注意使用 rawProcessFactory 创建的进程可能受 ProcessManager的控制, 会在关闭时候自动关闭所有由其创建的子进程

创建自己的IDE 技术实现 问题 教程

注解

此篇幅所有内容, 基于Theia当前版本:

% yarn list --pattern=@theia/core
yarn list v1.22.17
└─ @theia/core@1.42.1
✨  Done in 1.06s.
