Theia

一个IDE框架, 前后端分离的架构

以插件的形式编写代码, 支持依赖注入

后端可选浏览器/Electron

注意使用 rawProcessFactory 创建的进程可能受 ProcessManager的控制, 会在关闭时候自动关闭所有由其创建的子进程
