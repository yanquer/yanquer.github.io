animation 属性用来指定一组或多组动画，每组之间用逗号相隔。

animation 属性是 animation-name，animation-duration, animation-timing-function， animation-delay，animation-iteration-count，animation-direction，animation-fill-mode 和 animation-play-state 属性的一个简写属性形式

使用此属性, 就不必为了动效写很复杂的JS代码.

CSS3 @keyframes 规则

要创建 CSS3 动画，你需要了解 @keyframes 规则。

  • @keyframes 规则用来创建动画。
  • @keyframes 规则内指定一个 CSS 样式和动画将逐步从目前的样式更改为新的样式。

实例

@keyframes myfirst
{
    from {background: red;}
    to {background: yellow;}
}

@-webkit-keyframes myfirst /* Safari 与 Chrome */
{
    from {background: red;}
    to {background: yellow;}
}

使用 @keyframes 创建动画， 必须把它绑定到一个选择器，否则不会有任何效果。

指定至少这两个CSS3的动画属性绑定向一个选择器：

  • 规定动画的名称
  • 规定动画的时长

实例, 把 "myfirst" 动画捆绑到 div 元素，时长：5 秒：

div
{
    animation: myfirst 5s;
    -webkit-animation: myfirst 5s; /* Safari 与 Chrome */
}

注解

您必须定义动画的名称和动画的持续时间。如果省略的持续时间，动画将无法运行，因为默认值是0。

CSS3动画

动画是使元素从一种样式逐渐变化为另一种样式的效果。 您可以改变任意多的样式任意多的次数。

请用 百分比 来规定变化发生的时间，或用关键词 from 和 to，等同于 0% 和 100% 。

  • 0% 是动画的开始
  • 100% 是动画的完成。

为了得到最佳的浏览器支持，应始终定义 0% 和 100% 选择器。

实例: 当动画为 25% 及 50% 时改变背景色，然后当动画 100% 完成时再次改变：

@keyframes myfirst
{
    0%   {background: red;}
    25%  {background: yellow;}
    50%  {background: blue;}
    100% {background: green;}
}

@-webkit-keyframes myfirst /* Safari 与 Chrome */
{
    0%   {background: red;}
    25%  {background: yellow;}
    50%  {background: blue;}
    100% {background: green;}
}

CSS3的动画属性

使用格式:

animation: name duration timing-function delay iteration-count direction;

默认值:

animation: none 0 ease 0 1 normal

下面的表格列出了 @keyframes 规则和所有动画属性

属性 描述 CSS
@keyframes 规定动画。 3
animation 所有动画属性的简写属性。 3
animation-name 规定 @keyframes 动画的名称。 3
animation-duration 规定动画完成一个周期所花费的秒或毫秒。默认是 0。 3
animation-timing-function

规定动画的速度曲线。默认是 "ease"。

常用的时间函数

  • ease 表示动画或过渡会以较慢的速度开始,然后逐渐加快,在中间达到最快速度, 最后又逐渐减慢,形成一个平滑的s型曲线。 相当于使用 cubic-bezier(0.25, 0.1, 0.25, 1) 这个三次贝塞尔曲线作为时间函数。 这种速度变化模拟了现实世界中物体的运动特征,看起来更自然流畅。
  • ease-in 表示加速运动
  • ease-out 表示减速运动
  • ease-in-out 表示先加速后减速
  • linear 表示匀速运动
 3
animation-fill-mode 规定当动画不播放时（当动画完成时，或当动画有一个延迟未开始播放时），要应用到元素的样式。 3
animation-delay

规定动画何时开始。默认是 0。

支持的值

  • 设置为正数时,比如 animation-delay: 3s;，动画会在 3 秒钟后开始播放。 这意味着在动画开始之前,元素会保持静止状态 3 秒钟。
  • 设置为负数时,比如 animation-delay: -3s;， 动画会立即开始播放,但会从动画序列的中间位置开始。 负数值表示动画会"向后"播放 3 秒钟, 这样可以让动画看起来好像是从前一个周期结束的地方继续播放。

负数值主要是为了实现让动画立即(从上一个周期的某个位置)开始播放

 3
animation-iteration-count 规定动画被播放的次数。默认是 1。 3
animation-direction 规定动画是否在下一周期逆向地播放。默认是 "normal"。 3
animation-play-state 规定动画是否正在运行或暂停。默认是 "running"。 3

下面两个例子设置所有动画属性：

实例1: 运行myfirst动画，设置所有的属性：

div
{
    animation-name: myfirst;
    animation-duration: 5s;
    animation-timing-function: linear;
    animation-delay: 2s;
    animation-iteration-count: infinite;
    animation-direction: alternate;
    animation-play-state: running;
    /* Safari 与 Chrome: */
    -webkit-animation-name: myfirst;
    -webkit-animation-duration: 5s;
    -webkit-animation-timing-function: linear;
    -webkit-animation-delay: 2s;
    -webkit-animation-iteration-count: infinite;
    -webkit-animation-direction: alternate;
    -webkit-animation-play-state: running;
}

实例2, 与上面的动画相同，但是使用了简写的动画 animation 属性：

div
{
    animation: myfirst 5s linear 2s infinite alternate;
    /* Safari 与 Chrome: */
    -webkit-animation: myfirst 5s linear 2s infinite alternate;
}

实际使用实例

代码

<style>
  .waves-area .waves-svg {
    width: 100%;
    /* height: 80px; */
  }

  .parallax > use {
    animation: move-forever 30s cubic-bezier(0.55, 0.5, 0.45, 0.5) infinite;
  }

  .parallax > use:nth-child(1) {
    animation-delay: -2s;
    animation-duration: 7s;
    fill: var(--main-area-bg-color);
    opacity: 0.5;
  }

  .parallax > use:nth-child(2) {
    animation-delay: -3s;
    animation-duration: 10s;
    fill: var(--main-area-bg-color);
    opacity: 0.6;
  }

  .parallax > use:nth-child(3) {
    animation-delay: -4s;
    animation-duration: 13s;
    fill: var(--main-area-bg-color);
    opacity: 0.7;
  }

  .parallax > use:nth-child(4) {
    animation-delay: -5s;
    animation-duration: 20s;
    fill: var(--main-area-bg-color);
  }

  @keyframes move-forever {
    0% {
      transform: translate3d(-90px, 0, 0);
    }
    100% {
      transform: translate3d(85px, 0, 0);
    }
  }
</style>

<div class="waves-area">
  <section class="main-hero-waves-area waves-area">
    <svg class="waves-svg" preserveAspectRatio="none" shape-rendering="auto" viewBox="0 24 150 28"
        xlink="http://www.w3.org/1999/xlink" xmlns="http://www.w3.org/2000/svg">
      <defs>
        <path
            d="M -160 44 c 30 0 58 -18 88 -18 s 58 18 88 18 s 58 -18 88 -18 s 58 18 88 18 v 44 h -352 Z"
            id="gentle-wave"></path>
      </defs>
      <g class="parallax">
        <use href="#gentle-wave" x="48" y="0"></use>
        <use href="#gentle-wave" x="48" y="3"></use>
        <use href="#gentle-wave" x="48" y="5"></use>
        <use href="#gentle-wave" x="48" y="7"></use>
      </g>
    </svg>
  </section>
</div>
move-forever 30s cubic-bezier(0.55, 0.5, 0.45, 0.5) infinite; 的含义:

  • 定义的 move-forever 为 transform: translate3d(-90px, 0, 0);. 含义:

    • x 轴方向的位移量: -90px
    • y 轴方向的位移量: 0
    • z 轴方向的位移量: 0

    这个变换是在三维空间中进行的,即使 z 轴方向的位移量为 0, 也会使用 3D 变换引擎来执行这个变换,而不是使用 2D 变换。

  • 30s: 这个数值表示动画的播放时长为 30 秒。

  • cubic-bezier(0.55, 0.5, 0.45, 0.5): 这是定义了动画的时间函数(timing function)。

    cubic-bezier 是一种常见的时间函数类型,它使用四个数值来定义一个三次贝塞尔曲线。这个曲线决定了动画在时间轴上的变化速度。

    • 第一个和第四个数值范围在 0 到 1 之间,控制曲线的起始和结束点。
    • 第二个和第三个数值也在 0 到 1 之间,控制曲线的形状。

    在这个例子中, (0.55, 0.5, 0.45, 0.5) 定义了一个相对平滑的曲线,动画会先加速后减速。

  • infinite: 这个属性指定了动画播放的次数, infinite 表示动画会无限循环播放。
