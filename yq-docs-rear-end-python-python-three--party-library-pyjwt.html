pyjwt库 是 JSON Web Token 的Python实现.

pip install pyjwt
# pip install PyJWT

PyJWT的基本概念

  • 令牌（Token）：JWT的编码字符串，用于在网络应用之间传输信息。
  • 头部（Header）：包含令牌的元数据，通常包含算法和令牌类型。
  • 载荷（Payload）：包含声明的部分，例如用户ID、权限等。
  • 签名（Signature）：用于验证令牌的完整性，防止被篡改。
  • 算法（Algorithm）：用于生成和验证签名的算法，常见的有HS256、RS256等。

简单使用

import jwt
import datetime

# 定义密钥和有效载荷
secret_key = "your_secret_key"
payload = {
"user_id": 123,
"username": "example_user",
"exp": datetime.datetime.utcnow() + datetime.timedelta(minutes=5) # 5分钟后过期
}

# 生成JWT
token = jwt.encode(payload, secret_key, algorithm="HS256")
print(f"生成的JWT: {token}")

# 解码JWT
try:
decoded_payload = jwt.decode(token, secret_key, algorithms=["HS256"])
print(f"解码后的内容: {decoded_payload}")
except jwt.ExpiredSignatureError:
print("Token已过期")
except jwt.InvalidTokenError:
print("无效的Token")

为了增强安全性，可以在 JWT 中添加 exp（过期时间）和 nbf（生效时间）字段

payload = {
"user_id": 123,
"username": "example_user",
"exp": datetime.datetime.utcnow() + datetime.timedelta(seconds=60), # 1分钟后过期
"nbf": datetime.datetime.utcnow() + datetime.timedelta(seconds=10) # 10秒后生效
}

token = jwt.encode(payload, secret_key, algorithm="HS256")
print(f"带有过期和生效时间的JWT: {token}")

在分布式系统中，可以使用非对称加密算法（如 RS256）提高安全性

from jwt import encode, decode

# 加载公钥和私钥
with open("private_key.pem", "r") as f:
private_key = f.read()
with open("public_key.pem", "r") as f:
public_key = f.read()

# 使用私钥生成JWT
payload = {"user_id": 123, "username": "example_user"}
token = encode(payload, private_key, algorithm="RS256")

# 使用公钥验证JWT
decoded_payload = decode(token, public_key, algorithms=["RS256"])
print(f"解码后的内容: {decoded_payload}")

在实际开发中，JWT 通常用于用户认证系统。例如，将生成的 JWT 放入请求头的 Authorization 字段中:

Authorization: JWT <token>

在服务端，可以通过解码和验证 JWT 来实现用户身份认证。

封装的例子

一个封装的例子

# coding: utf-8

import jwt
import time
from datetime import datetime, timedelta


def generate_jwt(service_account_file, scopes):
    """
    生成用于 Google API 认证的 JWT

    Args:
        service_account_file: Service Account JSON 文件路径
        scopes: API 权限范围列表

    Returns:
        str: 生成的 JWT token
    """
    # 从 JSON 文件加载服务账号信息
    import json
    with open(service_account_file, 'r') as f:
        service_account_info = json.load(f)

    # JWT 配置
    now = int(time.time())
    payload = {
        'iss': service_account_info['client_email'],
        'scope': ' '.join(scopes),
        'aud': 'https://oauth2.googleapis.com/token',
        'exp': now + 3600,  # 1小时过期
        'iat': now
    }

    # 使用私钥签名
    private_key = service_account_info['private_key']

    # 生成 JWT
    jwt_token = jwt.encode(
        payload,
        private_key,
        algorithm='RS256',
        headers={'alg': 'RS256', 'typ': 'JWT'}
    )

    return jwt_token


if __name__ == '__main__':
    SERVICE_ACCOUNT_FILE = 'common/business/admob/web-client-auth2.json'

    SCOPES = ['https://www.googleapis.com/auth/admob.readonly']

    jwt_token = generate_jwt(SERVICE_ACCOUNT_FILE, SCOPES)

    print(jwt_token)
