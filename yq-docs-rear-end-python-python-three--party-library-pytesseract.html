pytesseract是一个基于Tesseract-OCR的OCR工具, 可以用于文字识别, 支持多种语言, 包括中文, 英文, 日文等.

安装:

pip install pytesseract

默认情况下不支持中文识别, 如果需要识别中文, 需要安装中文语言包, 并指定语言:

pip install pytesseract
sudo apt-get install tesseract-ocr
sudo apt-get install tesseract-ocr-chi-sim

一个简单的使用例子:

import pytesseract
from PIL import Image

img = Image.open('example.jpg')
text = pytesseract.image_to_string(img, lang='chi_sim')

print(text)

预处理

在进行 OCR 识别之前，有时需要对图像进行预处理，以提高识别率。以下是一个简单的图像预处理示例:

import cv2
import numpy as np

# 使用 OpenCV 读取图像
image_cv = cv2.imread('sample.png')

# 转为灰度图
gray_image = cv2.cvtColor(image_cv, cv2.COLOR_BGR2GRAY)

# 应用二值化处理
_, binary_image = cv2.threshold(gray_image, 150, 255, cv2.THRESH_BINARY)

# 使用 pytesseract 识别处理后的图像
text_processed = pytesseract.image_to_string(binary_image, lang='eng')
print('处理后的识别文本：', text_processed)

例二:

import pytesseract
from PIL import Image

save_path = 'test.png'
image = cv2.imread(save_path)

# 将图片转换为灰度图
gray = cv2.cvtColor(image, cv2.COLOR_BGR2GRAY)

# 应用阈值化以突出显示白色文字
_, thresh = cv2.threshold(gray, 200, 255, cv2.THRESH_BINARY_INV)

# 保存处理后的图像（可选）
processed_image_path = save_path
cv2.imwrite(processed_image_path, thresh)

# 使用Pillow加载处理后的图像
processed_image = Image.open(processed_image_path)

# 使用pytesseract将图片转换为文字
text = pytesseract.image_to_string(processed_image)

多语言

此py库默认支持语言:

['eng', 'osd', 'snum']

如果需要识别其他语言, 需要完整安装 tesseract , 设置testdata

比如在windows下

# Example config: r'--tessdata-dir "C:\Program Files (x86)\Tesseract-OCR\tessdata"'
# It's important to add double quotes around the dir path.
tessdata_dir_config = r'--tessdata-dir "<replace_with_your_tessdata_dir_path>"'
pytesseract.image_to_string(image, lang='chi_sim', config=tessdata_dir_config)

在Mac下安装语言包:

brew install tesseract-lang

tesseract 目录可以通过命令获取:

brew info tesseract
# /usr/local/Cellar/tesseract-lang/4.1.0/
# /usr/local/Cellar/tesseract-lang/4.1.0/share/tessdata/

使用:

pytesseract.image_to_string(image, lang='chi_sim', config='--tessdata-dir "/usr/local/Cellar/tesseract-lang/4.1.0/share/tessdata/"')

注解

还可以在 https://github.com/tesseract-ocr/tessdata 手动下载语言包文件夹(只下载需要的), 然后指定路径

CLI工具

Mac下安装:

brew install tesseract tesseract-lang

其中, tesseract-lang 是综合语言包

使用:

tesseract example.jpg output -l chi_sim

# 输出到控制台
tesseract example.jpg stdout -l chi_sim

# 输出到txt
tesseract example.jpg output -l chi_sim txt

# 输出到hocr
tesseract example.jpg output -l chi_sim hocr

# 输出到pdf
tesseract example.jpg output -l chi_sim pdf

# 输出到tsv
tesseract example.jpg output -l chi_sim tsv

# 输出到box
tesseract example.jpg output -l chi_sim box

# 输出到alto
tesseract example.jpg output -l chi_sim alto
