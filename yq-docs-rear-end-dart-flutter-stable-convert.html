convert

修改于: 2024-11-24
参考
  • JSON 和序列化数据

引入:

dart:convert

此库可用来辅助 JSON 序列化

JSON序列化

用例1-简单解码

比如有以下数据

{
  "name": "John Smith",
  "email": "john@example.com"
}

简单的解码为对象

final user = jsonDecode(jsonString) as Map<String, dynamic>;

print('Howdy, ${user['name']}!');
print('We sent the verification link to ${user['email']}.');

用例2-模型类序列化

模型类需要实现的 实例函数

  • 类.fromJson() 构造函数，用于从映射中构造一个新的 类 实例。
  • toJson() 方法，这个方法会将 类 实例转换为一个映射。

以下以一个 User 类举例说明

class User {
  final String name;
  final String email;

  User(this.name, this.email);

  User.fromJson(Map<String, dynamic> json)
      : name = json['name'] as String,
        email = json['email'] as String;

  Map<String, dynamic> toJson() => {
        'name': name,
        'email': email,
      };
}

解码逻辑的责任现在转移到了模型内部。通过这个新方法，可以很容易地解码获得一个 user 实例。

// 获取map字典
final userMap = jsonDecode(jsonString) as Map<String, dynamic>;
// 构造实例
final user = User.fromJson(userMap);

print('Howdy, ${user.name}!');
print('We sent the verification link to ${user.email}.');

编码为字符串

String json = jsonEncode(user);

注解

不需要主动调用 toJson() 方法，因为 jsonEncode() 已经帮你做了这件事。
