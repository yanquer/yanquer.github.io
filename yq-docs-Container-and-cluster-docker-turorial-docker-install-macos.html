docker安装MacOS

修改于: 2024-07-09

参考: https://mp.weixin.qq.com/s/nBxSM2RGyuNA_VdnE31ueg

docker-compose.yml配置:

version: '3'
services:
  docker-osx:
    image: sickcodes/docker-osx:latest
    devices:
      - "/dev/kvm"
    ports:
      - "50922:10022"
    volumes:
      - "/tmp/.X11-unix:/tmp/.X11-unix"
    environment:
      - "DISPLAY=${DISPLAY:-:0.0}

支持不同版本(参照: https://hub.docker.com/r/sickcodes/docker-osx )

镜像 对应Mac大版本 大小
docker-osx:latest Catalina 1.72GB
docker-osx:big-sur Big Sur 2.31GB
docker-osx:monterey Monterey 2.3GB
docker-osx:ventura Ventura 2.36GB
docker-osx:auto Catalina Pre-Installed 21.2GB
