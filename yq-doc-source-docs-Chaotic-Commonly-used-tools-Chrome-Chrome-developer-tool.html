Chrome开发者工具

一般默认 F12 即可打开

标签

Elements

此界面显示 HTML 的渲染内容, 可 Ctrl/Cmd + F 在当前页面搜索

侧边还有样式显示, 计算显示

Console

控制台, 在此界面

可 Ctrl/Cmd + Shift + F 全局搜索字符串;

Ctrl/Cmd + O 或者 Ctrl/Cmd + P，可以呼出一个文件列表窗口， 输入关键词，可以搜索到 文件名包含这个词的文件

Network
网络连接

一些快捷键

Ctrl/Cmd + Shift + P
打开一个运行命令的窗口; 可直接上下键选择打开哪一个标签页
Esc

显示/隐藏当前标签页抽屉栏, 只在控制台跟源代码标签页用过.
