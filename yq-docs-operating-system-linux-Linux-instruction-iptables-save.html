iptables-save

修改于: 2023-02-20

批量导出Linux防火墙规则，

直接执行：显示当前启用的所有规则，按raw、mangle、nat、filter顺序列出

-c 指定在还原iptables表时候，还原当前的数据包计数器和字节计数器的值；
-t

指定表

“#”号开头的表示注释；

“*表名”表示所在的表；

“：链名默认策略”表示相应的链及默认策略，具体的规则部分省略了命令名“iptables”；

“COMMIT”表示提交前面的规则设置；

注解

备份恢复时， iptables-save、 iptables 两个都需要搭配重定向符使用
