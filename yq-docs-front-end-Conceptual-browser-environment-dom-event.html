鼠标事件
  • mousedown: 鼠标按下事件
  • mousemove: 鼠标移动事件
  • mouseup: 鼠标抬起事件
  • mouseover: 只要鼠标指针移入事件所绑定的元素或其子元素，都会触发该事件
  • mouseenter: 只有鼠标指针移入事件所绑定的元素时，才会触发该事件
  • mouseout: 只要鼠标指针移出事件所绑定的元素或其子元素，都会触发该事件
  • mouseleave: 只有鼠标指针移出事件所绑定的元素时，才会触发该事件

问题 - mouseleave / mouseenter 失效

参考: CSDN 鼠标在子元素内移动时，任然触发了父元素mouseleave事件，个人偶遇情况

理论上, mouseenter 和 mouseleave 事件只会在鼠标进入或离开元素本身时触发，而不会在子元素上触发。

有时候可能会遇到失效的情况, 比如, 背景元素部分区域的 z-index 值比当前元素高, 且这部分元素视觉上是属于背景的. 导致鼠标事件进入了背景元素, 这时就会导致 mouseenter 和 mouseleave 事件触发而失效。

解决方案, 要么将背景部分的 z-index 值设置为比当前元素低, 要么将当前元素的 z-index 值设置为比背景部分高.
