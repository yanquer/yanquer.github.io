overflow

修改于: 2024-03-25

使用不同的overflow值来实现垂直滚动条、水平滚动条，或者两者都有

支持的值

overflow: initial:

将 overflow 属性设置为其初始值。 初始值取决于元素的 display 属性:

  • 对于 display: block 元素,初始值为 visible 。
  • 对于 display: inline 元素,初始值为 auto 。

使用 initial 可以将元素的 overflow 属性重置为默认值,而不受父元素的影响。

overflow: inherit:

将 overflow 属性继承自父元素。 子元素的 overflow 属性会继承父元素的设置,而不管子元素自身的 display 属性。

使用 inherit 可以让子元素完全继承父元素的 overflow 属性值。

overflow: auto overflow: hidden

垂直滚动条 overflow-y

当内容超出div的垂直空间时，显示垂直滚动条。

.div-class {
  overflow-y: scroll; /* 只有当内容溢出时才显示垂直滚动条 */
}

或者，如果你想要在内容溢出时自动显示滚动条，并且当内容没有溢出时也不显示滚动条，可以使用auto：

.div-class {
  overflow-y: auto;
}

水平滚动条 overflow-x

当内容超出div的水平空间时，显示水平滚动条。

.div-class {
  overflow-x: scroll; /* 只有当内容溢出时才显示水平滚动条 */
}

或者使用auto：

.div-class {
  overflow-x: auto;
}

同时添加垂直和水平滚动条 overflow

如果你想让div在内容溢出时同时显示垂直和水平滚动条，可以这样设置：

.div-class {
  overflow: scroll; /* 同时显示垂直和水平滚动条 */
}

使用auto：

.div-class {
  overflow: auto;
}

隐藏滚动条，但仍然允许滚动

如果你想要隐藏滚动条，但仍然允许通过鼠标滚轮或其他方式进行滚动，可以设置overflow属性为hidden：

.div-class {
  overflow: hidden;
}

注解

overflow: hidden; 将不允许内容溢出div，滚动条也不会显示。
