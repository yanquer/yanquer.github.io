用于集成keycloak的JavaScript适配器

此模块需要先安装配置好 keycloak 服务器

安装:

# 全局安装
npm install keycloak-js

使用

import Keycloak, { KeycloakConfig } from 'keycloak-js';


let keycloakConfig: KeycloakConfig | undefined = undefined;
const keycloak = new Keycloak(keycloakConfig);

keycloak.onReady = (authenticated) => { console.log('onReady', authenticated); };
keycloak.onAuthSuccess = () => { console.log('onAuthSuccess'); };
keycloak.onAuthError = (errorData) => { console.error('onAuthError', errorData); };
keycloak.onAuthRefreshSuccess = () => { console.log('onAuthRefreshSuccess'); };
keycloak.onAuthRefreshError = () => { console.log('onAuthRefreshError'); };
keycloak.onAuthLogout = () => { console.log('onAuthLogout'); };
keycloak.onTokenExpired = () => { console.log('onTokenExpired'); };

keycloak
  .init({
    onLoad: 'check-sso',
    // onLoad: 'login-required',
    redirectUri: window.location.href,
    checkLoginIframe: false
  })
  .then(auth => {
    if (auth) {
      console.log(`auth ${auth}`)
      const parsedToken = keycloak.idTokenParsed;
      if (parsedToken) {
        const userMail = parsedToken.email;
        setToken(keycloak.idToken);
        setEmail(userMail);
        setLogoutUrl(keycloak.createLogoutUrl());
      }
    }
  })
  .catch((e) => {
    console.error('Authentication Failed');
    console.error(e)
  });

一般会在页面初始化的时候使用 check-sso 模式检查是否登录, 需要登陆时, 使用 login-required 模式, 登录成功后, 会自动跳转到 redirectUri 指定的页面.
