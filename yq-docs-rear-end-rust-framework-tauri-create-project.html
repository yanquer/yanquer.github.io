创建tauri项目

修改于: 2025-03-01
相关资源
  • tauri文档-创建项目: https://tauri.app/zh-cn/start/create-project/

近段时间使用前端技术比较习惯, 所以采用前端的形式创建:

yarn create tauri-app
# npm 可以: npm create tauri-app@latest

示例

$ yarn create tauri-app

(Use `node --trace-deprecation ...` to show where the warning was created)

进入项目后安装必要的依赖项

yarn install

转换为nextjs项目

从上面的创建流程创建的默认是vite的前端应用, 目前的脚手架并没有提供 nextjs 的选项, 所以只能手动转了.

进去项目目录, 安装相关包

yarn add next@latest react@latest react-dom@latest

更新 Tauri 配置 src-tauri/tauri.conf.json

{
  "build": {
      "beforeDevCommand": "yarn dev",
      "beforeBuildCommand": "yarn build",
      "devUrl": "http://localhost:3000",
      "frontendDist": "../dist"
  }
}

注解

偶然发现一个开源模版 https://github.com/kvnxiao/tauri-nextjs-template/
